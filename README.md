## 安装方法

主要用于视频的使用，语言是 ruby

需要安装 ffmpeg。并添加环境变量

按 <kbd>Win</kbd> + <kbd>R</kbd> -> 运行输入 `sysdm.cpl` <kbd>Enter</kbd> -> 高级 -> 环境变量

上方的 Path 双击 -> 新建 -> 输入你的 ffmpeg 的 bin 文件夹全路径 即可。

比如我的是 `E:\worksofts\__media\ffmpeg\bin`
